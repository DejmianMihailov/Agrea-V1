## ✨ Анимации и ефекти

### Hover ефекти:
- **Бутони:** Потъмняват, показват ripple ефект
- **Карти:** Издигат се (translateY), сянка се увеличава
- **Линкове:** Подчертават се със зелена линия
- **Категории:** Завъртане на иконата, scale up

### Scroll анимации:
- **Секции:** Fade in + translateY при влизане в viewport
- **Smooth scroll:** При навигация между секции
- **Back to top:** Показва се след 300px scroll

### Loading states:
- **Формуляри:** Spinner при submit
- **Изображения:** Fade in при зареждане
